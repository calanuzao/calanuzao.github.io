Audio & DSP Projects

Noise Bliss

An advanced acoustic scene analysis project focused on urban sound event detection. The system leverages deep learning architectures (CNN and TCN) to classify environmental sounds from the SONYC Urban Sound Tagging (SONYC-UST) dataset. This project represents a complete pipeline for audio processing, feature extraction, model training, and urban noise classification.

  • Advanced noise generation algorithms
  • Real-time audio processing capabilities
  • Cross-platform compatibility
  • Modular design for easy extension

VSS (Virtual Sound System)

This project implements a Virtual Surround Sound (VSS) system using the SADIE HRTF database. The system processes 5.1 surround audio to create both a binaural output for headphone listening and a standard stereo downmix.

  • 3D spatial audio rendering
  • Room acoustics simulation
  • HRTF processing capabilities
  • Low-latency audio processing pipeline

Hearing Aid EQ

Simulation of Bilateral Sudden Sensorineural Hearing Loss (SSNHL) in MATLAB for Physical Acoustics in Directional Hearing.
